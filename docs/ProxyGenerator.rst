
ProxyGenerator module
-----------------------

Only the SOCKS5 proxy workflows are recommended:
``Socks5Proxies()``, ``Socks5ProxyFile()``, and
``scholarly.load_socks5_proxy_file()``. The other proxy helpers are kept for
backward compatibility and are deprecated.
